Fast made surveys platform


Features:
- Add surveys
- Limit surveys reactions based on IP Address
- Modify new survey attributes
- Sync with MySQL database 


How to run:
- main folder:
- enter command: node api/api.js
- Edit MySQL settings in the first lines (api.js)
- npm run dev
